# Sistema de Gestión de Biblioteca v1.2

## Descripción
Proyecto desarrollado en **Java** que implementa un sistema de gestión para préstamos, devoluciones y registros de usuarios y libros.

## Conceptos de POO Aplicados
- **Abstracción:** Uso de clases abstractas para la base de usuarios.
- **Herencia:** Especialización de perfiles (Estudiante y Profesor).
- **Polimorfismo:** Reglas de préstamo dinámicas según el tipo de usuario.
- **Encapsulamiento:** Protección de datos y lógica de negocio centralizada.

## Tecnologías
- Java JDK 17+
- Persistencia en archivos CSV.
